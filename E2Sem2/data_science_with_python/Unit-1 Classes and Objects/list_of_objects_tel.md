## 2Ô∏è‚É£ Quick Recall: The Student Class


"‡∞Æ‡∞® ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç ‡∞ï‡±ç‡∞≤‡∞æ‡∞∏‡±ç ‡∞®‡∞ø ‡∞Æ‡∞≥‡±ç‡∞≥‡±Ä ‡∞í‡∞ï‡∞∏‡∞æ‡∞∞‡∞ø ‡∞ó‡±Å‡∞∞‡±ç‡∞§‡±Å‡∞ö‡±á‡∞∏‡±Å‡∞ï‡±Å‡∞Ç‡∞¶‡∞æ‡∞Ç."


```python
class Student:
   def __init__(self, name="Unknown", student_id=-1):
       self.name = name
       self.student_id = student_id


   def show_identity(self):
       print(f"Name: {self.name} | ID: {self.student_id}")
```

## 5Ô∏è‚É£ MEMORY VISUALIZATION (List of Objects)

 * `students` ‡∞Ö‡∞®‡±á‡∞¶‡∞ø ‡∞ï‡±á‡∞µ‡∞≤‡∞Ç ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞≤‡±Å ‡∞∞‡∞æ‡∞∏‡±Å‡∞ï‡±Å‡∞®‡±ç‡∞® ‡∞í‡∞ï ‡∞™‡±á‡∞™‡∞∞‡±ç (List) ‡∞Æ‡∞æ‡∞§‡±ç‡∞∞‡∞Æ‡±á.

## 6Ô∏è‚É£ COMMON MISTAKES 


## 1. The "One Object" Mistake 



 * **Result:** ‡∞≤‡∞ø‡∞∏‡±ç‡∞ü‡±ç ‡∞≤‡±ã 5 ‡∞ê‡∞ü‡∞Æ‡±ç‡∞∏‡±ç ‡∞â‡∞Ç‡∞ü‡∞æ‡∞Ø‡∞ø, ‡∞ï‡∞æ‡∞®‡±Ä ‡∞Ö‡∞µ‡∞®‡±ç‡∞®‡±Ä **‡∞í‡∞ï‡±á ‡∞í‡∞ï‡±ç‡∞ï** ‡∞Æ‡±Ü‡∞Æ‡∞∞‡±Ä ‡∞¨‡∞æ‡∞ï‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞ö‡±Ç‡∞™‡∞ø‡∞∏‡±ç‡∞§‡∞æ‡∞Ø‡∞ø. ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞í‡∞ï‡±ç‡∞ï‡∞∞‡∞ø ‡∞™‡±á‡∞∞‡±Å ‡∞Æ‡∞æ‡∞∞‡∞ø‡∞∏‡±ç‡∞§‡±á, **‡∞Ö‡∞Ç‡∞¶‡∞∞‡∞ø ‡∞™‡±á‡∞∞‡±ç‡∞≤‡±Å ‡∞Æ‡∞æ‡∞∞‡∞ø‡∞™‡±ã‡∞§‡∞æ‡∞Ø‡∞ø**.
 * **Fix:** `s = Student()` ‡∞Ö‡∞®‡±á‡∞¶‡∞ø ‡∞ï‡∞ö‡±ç‡∞ö‡∞ø‡∞§‡∞Ç‡∞ó‡∞æ Loop **‡∞≤‡±ã‡∞™‡∞≤‡±á** ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø.


## 2. List ‡∞®‡∞ø Object ‡∞Ö‡∞®‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞Ç
