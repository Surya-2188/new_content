## ‚≠ê 1Ô∏è‚É£ ‡∞Ö‡∞∏‡∞≤‡±Å ‡∞∏‡∞Æ‡∞∏‡±ç‡∞Ø ‡∞è‡∞Ç‡∞ü‡∞ø? (Real-World Problem)


‡∞Æ‡∞®‡∞Ç ‡∞í‡∞ï **‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç ‡∞Æ‡±á‡∞®‡±á‡∞ú‡±ç‚Äå‡∞Æ‡±Ü‡∞Ç‡∞ü‡±ç ‡∞∏‡∞ø‡∞∏‡±ç‡∞ü‡∞Æ‡±ç** ‡∞§‡∞Ø‡∞æ‡∞∞‡±Å ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞®‡±ç‡∞®‡∞æ‡∞Ç ‡∞Ö‡∞®‡±Å‡∞ï‡±ã‡∞Ç‡∞°‡∞ø.
‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç ‡∞Ü‡∞¨‡±ç‡∞ú‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞≤‡±ã ‡∞á‡∞µ‡∞ø ‡∞â‡∞Ç‡∞ü‡∞æ‡∞Ø‡∞ø:


 * `name` (‡∞™‡±á‡∞∞‡±Å)
 * `marks` (‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å)
 * `section` (‡∞∏‡±Ü‡∞ï‡±ç‡∞∑‡∞®‡±ç)


‡∞á‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å ‡∞í‡∞ï ‡∞ü‡±Ä‡∞ö‡∞∞‡±ç ‡∞ï‡∞ø ‡∞á‡∞≤‡∞æ‡∞Ç‡∞ü‡∞ø ‡∞∏‡∞Ç‡∞¶‡±á‡∞π‡∞æ‡∞≤‡±Å ‡∞∞‡∞æ‡∞µ‡∞ö‡±ç‡∞ö‡±Å:


**Q1: "‡∞à ‡∞á‡∞¶‡±ç‡∞¶‡∞∞‡±Å ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç ‡∞í‡∞ï‡±ç‡∞ï‡∞∞‡±á‡∞®‡∞æ?"**


 * *‡∞Ö‡∞∞‡±ç‡∞•‡∞Ç:* ‡∞á‡∞¶‡±ç‡∞¶‡∞∞‡∞ø ‡∞™‡±á‡∞∞‡±Å, ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å, ‡∞∏‡±Ü‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞í‡∞ï‡±ç‡∞ï‡∞ü‡±á‡∞®‡∞æ?
 * *‡∞ï‡±ã‡∞°‡±ç ‡∞≤‡±ã ‡∞é‡∞≤‡∞æ ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø:* `if s1 == s2:`


**Q2: "‡∞é‡∞µ‡∞∞‡±Å ‡∞ü‡∞æ‡∞™‡∞∞‡±ç?"**


 * *‡∞Ö‡∞∞‡±ç‡∞•‡∞Ç:* ‡∞í‡∞ï‡∞∞‡∞ø ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å ‡∞Æ‡∞∞‡±ä‡∞ï‡∞∞‡∞ø ‡∞ï‡∞Ç‡∞ü‡±á ‡∞é‡∞ï‡±ç‡∞ï‡±Å‡∞µ ‡∞â‡∞®‡±ç‡∞®‡∞æ‡∞Ø‡∞æ?
 * *‡∞ï‡±ã‡∞°‡±ç ‡∞≤‡±ã ‡∞é‡∞≤‡∞æ ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø:* `if s1 > s2:`


**Q3: "‡∞ó‡±ç‡∞∞‡±Ç‡∞™‡±ç ‡∞™‡±ç‡∞∞‡∞æ‡∞ú‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞á‡∞¶‡±ç‡∞¶‡∞∞‡∞ø‡∞®‡±Ä ‡∞ï‡∞≤‡∞™‡∞µ‡∞ö‡±ç‡∞ö‡∞æ?"**


 * *‡∞Ö‡∞∞‡±ç‡∞•‡∞Ç:* ‡∞ü‡±Ä‡∞Æ‡±ç ‡∞∏‡±ç‡∞ï‡±ã‡∞∞‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞á‡∞¶‡±ç‡∞¶‡∞∞‡∞ø ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å ‡∞ï‡∞≤‡∞™‡∞æ‡∞≤‡∞ø.
 * *‡∞ï‡±ã‡∞°‡±ç ‡∞≤‡±ã ‡∞é‡∞≤‡∞æ ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø:* `team_score = s1 + s2`


**‡∞™‡∞æ‡∞§ ‡∞™‡∞¶‡±ç‡∞ß‡∞§‡∞ø (Java/C++):**
‡∞Æ‡∞®‡∞Ç ‡∞á‡∞≤‡∞æ ‡∞™‡±Ü‡∞¶‡±ç‡∞¶ ‡∞™‡±Ü‡∞¶‡±ç‡∞¶ ‡∞Æ‡±Ü‡∞•‡∞°‡±ç‡∞∏‡±ç ‡∞∞‡∞æ‡∞Ø‡∞æ‡∞≤‡±ç‡∞∏‡∞ø ‡∞µ‡∞ö‡±ç‡∞ö‡±á‡∞¶‡∞ø:


 * `s1.equals(s2)`
 * `s1.isGreaterThan(s2)`
 * `s1.addStudent(s2)`


**‡∞™‡±à‡∞•‡∞æ‡∞®‡±ç ‡∞™‡∞¶‡±ç‡∞ß‡∞§‡∞ø:**
‡∞™‡±à‡∞•‡∞æ‡∞®‡±ç ‡∞è‡∞Æ‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞Ç‡∞ü‡±á:


> *"‡∞Ö‡∞Ç‡∞ï‡±Ü‡∞≤‡±Å (Integers), ‡∞∏‡±ç‡∞ü‡±ç‡∞∞‡∞ø‡∞Ç‡∞ó‡±ç‡∞∏‡±ç (Strings) ‡∞µ‡∞æ‡∞°‡∞ø‡∞®‡∞ü‡±ç‡∞≤‡±Å‡∞ó‡∞æ‡∞®‡±á.. ‡∞Æ‡±Ä ‡∞Ü‡∞¨‡±ç‡∞ú‡±Ü‡∞ï‡±ç‡∞ü‡±ç‡∞∏‡±ç ‡∞ï‡∞ø ‡∞ï‡±Ç‡∞°‡∞æ `+`, `>`, `==` ‡∞ó‡±Å‡∞∞‡±ç‡∞§‡±Å‡∞≤‡∞®‡±Å ‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å ‡∞µ‡∞æ‡∞°‡∞ï‡±Ç‡∞°‡∞¶‡±Å?"*


‡∞¶‡±Ä‡∞®‡∞ø ‡∞ï‡±ã‡∞∏‡∞Æ‡±á **‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ú‡∞ø‡∞ï‡±ç ‡∞Æ‡±Ü‡∞•‡∞°‡±ç‡∞∏‡±ç (Magic Methods)** ‡∞≤‡±á‡∞¶‡∞æ **Dunder Methods** ‡∞µ‡∞ö‡±ç‡∞ö‡∞æ‡∞Ø‡∞ø.

## ‚≠ê 2Ô∏è‚É£ ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ú‡∞ø‡∞ï‡±ç ‡∞Æ‡±Ü‡∞•‡∞°‡±ç‡∞∏‡±ç ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞è‡∞Ç‡∞ü‡∞ø? (Intuition)


‡∞á‡∞µ‡∞ø ‡∞∏‡±ç‡∞™‡±Ü‡∞∑‡∞≤‡±ç ‡∞Æ‡±Ü‡∞•‡∞°‡±ç‡∞∏‡±ç. ‡∞µ‡±Ä‡∞ü‡∞ø ‡∞™‡±á‡∞∞‡±Å‡∞ï‡±Å ‡∞Æ‡±Å‡∞Ç‡∞¶‡±Å, ‡∞µ‡±Ü‡∞®‡±Å‡∞ï **‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞Ö‡∞Ç‡∞°‡∞∞‡±ç‚Äå‡∞∏‡±ç‡∞ï‡±ã‡∞∞‡±ç‡∞≤‡±Å (`__`)** ‡∞â‡∞Ç‡∞ü‡∞æ‡∞Ø‡∞ø.
‡∞Æ‡∞®‡∞Ç ‡∞Ü‡∞™‡∞∞‡±á‡∞ü‡∞∞‡±ç‡∞∏‡±ç ‡∞µ‡∞æ‡∞°‡∞ø‡∞®‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å, ‡∞™‡±à‡∞•‡∞æ‡∞®‡±ç ‡∞≤‡±ã‡∞™‡∞≤ ‡∞à ‡∞Æ‡±Ü‡∞•‡∞°‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞™‡∞ø‡∞≤‡±Å‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.


| ‡∞Æ‡∞®‡∞Ç ‡∞∞‡∞æ‡∞∏‡±á ‡∞ï‡±ã‡∞°‡±ç | ‡∞™‡±à‡∞•‡∞æ‡∞®‡±ç ‡∞≤‡±ã‡∞™‡∞≤ ‡∞™‡∞ø‡∞≤‡∞ø‡∞ö‡±á‡∞¶‡∞ø |
| :--- | :--- |
| `s1 == s2` | `s1.__eq__(s2)` |
| `s1 > s2` | `s1.__gt__(s2)` |
| `s1 + s2` | `s1.__add__(s2)` |
| `print(s1)` | `print(s1.__str__())` |


‡∞Æ‡∞®‡∞Ç ‡∞à ‡∞Æ‡±Ü‡∞•‡∞°‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞ï‡±ç‡∞≤‡∞æ‡∞∏‡±ç ‡∞≤‡±ã ‡∞°‡∞ø‡∞´‡±à‡∞®‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞æ‡∞®‡±ç‡∞®‡±á **Operator Overloading** ‡∞Ö‡∞Ç‡∞ü‡∞æ‡∞∞‡±Å.

## ‚≠ê 3Ô∏è‚É£ ‡∞™‡±Ç‡∞∞‡±ç‡∞§‡∞ø ‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£ ‚Äî ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç ‡∞ï‡±ç‡∞≤‡∞æ‡∞∏‡±ç


‡∞á‡∞ï‡±ç‡∞ï‡∞° ‡∞í‡∞ï ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç ‡∞ï‡±ç‡∞≤‡∞æ‡∞∏‡±ç, ‡∞¨‡∞ø‡∞≤‡±ç‡∞ü‡±ç-‡∞á‡∞®‡±ç ‡∞ü‡±à‡∞™‡±ç ‡∞≤‡∞æ‡∞ó‡∞æ ‡∞é‡∞≤‡∞æ ‡∞™‡±ç‡∞∞‡∞µ‡∞∞‡±ç‡∞§‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡±ã ‡∞ö‡±Ç‡∞°‡∞Ç‡∞°‡∞ø.


```python
class Student:
   def __init__(self, name, marks, section):
       self.name = name
       self.marks = marks
       self.section = section


   # ----------------------------------------------
   # 1. ‡∞∏‡∞Æ‡∞æ‡∞®‡∞Æ‡∞æ ‡∞ï‡∞æ‡∞¶‡∞æ? (Equality Check - ==)
   # ----------------------------------------------
   def __eq__(self, other):
       # ‡∞™‡±á‡∞∞‡±Å, ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å, ‡∞∏‡±Ü‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞Ö‡∞®‡±ç‡∞®‡±Ä ‡∞∏‡∞Æ‡∞æ‡∞®‡∞Æ‡±à‡∞§‡±á‡∞®‡±á 'True' ‡∞á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø
       return (self.name.lower() == other.name.lower() and
               self.marks == other.marks and
               self.section.lower() == other.section.lower())


   # ----------------------------------------------
   # 2. ‡∞é‡∞µ‡∞∞‡±Å ‡∞ó‡±ä‡∞™‡±ç‡∞™? (Greater Than - >)
   # ----------------------------------------------
   def __gt__(self, other):
       # ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å ‡∞é‡∞ï‡±ç‡∞ï‡±Å‡∞µ ‡∞â‡∞Ç‡∞ü‡±á ‡∞µ‡∞æ‡∞≥‡±ç‡∞≤‡±á ‡∞ó‡±ä‡∞™‡±ç‡∞™
       return self.marks > other.marks


   # ----------------------------------------------
   # 3. ‡∞ï‡∞≤‡∞™‡∞°‡∞Ç (Addition - +)
   # ----------------------------------------------
   def __add__(self, other):
       # ‡∞á‡∞¶‡±ç‡∞¶‡∞∞‡∞ø‡∞®‡±Ä ‡∞ï‡∞≤‡∞ø‡∞™‡∞ø ‡∞í‡∞ï '‡∞ï‡±ä‡∞§‡±ç‡∞§' ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç (‡∞ü‡±Ä‡∞Æ‡±ç) ‡∞®‡∞ø ‡∞∏‡±É‡∞∑‡±ç‡∞ü‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø
       new_name = self.name + " & " + other.name
       total_marks = self.marks + other.marks
       return Student(new_name, total_marks, self.section)


   # ----------------------------------------------
   # 4. ‡∞∏‡±ç‡∞ü‡±ç‡∞∞‡∞ø‡∞Ç‡∞ó‡±ç ‡∞≤‡∞æ‡∞ó‡∞æ ‡∞ö‡±Ç‡∞™‡∞ø‡∞Ç‡∞ö‡∞°‡∞Ç (print)
   # ----------------------------------------------
   def __str__(self):
       return f"{self.name} ({self.section}) - Marks: {self.marks}"


   # ----------------------------------------------
   # 5. ‡∞°‡±Ü‡∞µ‡∞≤‡∞™‡∞∞‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç (Debugging - repr)
   # ----------------------------------------------
   def __repr__(self):
       return f"Student('{self.name}', {self.marks}, '{self.section}')"


# --- ‡∞µ‡∞ø‡∞®‡∞ø‡∞Ø‡±ã‡∞ó‡∞Ç (USAGE) ---
s1 = Student("Raju", 85, "A")
s2 = Student("raju", 85, "a")  # ‡∞°‡±á‡∞ü‡∞æ ‡∞í‡∞ï‡±ç‡∞ï‡∞ü‡±á, ‡∞ï‡±á‡∞∏‡±ç (Case) ‡∞µ‡±á‡∞∞‡±Å
s3 = Student("Kiran", 90, "A")


# 1. Equality Check
print(f"Is s1 equal to s2? {s1 == s2}")  # True


# 2. Comparison
print(f"Is Kiran > Raju?   {s3 > s1}")  # True


# 3. Addition
team = s1 + s3
print(f"Team: {team}")  # Ravi&Kiran (A) - Marks: 175


# 4. Printing
print(s1)  # Ravi (A) - Marks: 85
```

## ‚≠ê 4Ô∏è‚É£ ‡∞≤‡±à‡∞®‡±ç-‡∞¨‡±à-‡∞≤‡±à‡∞®‡±ç ‡∞µ‡∞ø‡∞µ‡∞∞‡∞£


### `__eq__(self, other)`


 * **‡∞é‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å ‡∞∞‡∞®‡±ç ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø:** `s1 == s2` ‡∞Ö‡∞®‡±ç‡∞®‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å.
 * **‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç:** ‡∞™‡±á‡∞∞‡±Å (‡∞ö‡∞ø‡∞®‡±ç‡∞®/‡∞™‡±Ü‡∞¶‡±ç‡∞¶ ‡∞Ö‡∞ï‡±ç‡∞∑‡∞∞‡∞æ‡∞≤‡±Å ‡∞™‡∞ü‡±ç‡∞ü‡∞ø‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞ï‡±Å‡∞Ç‡∞°‡∞æ), ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å, ‡∞∏‡±Ü‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞Ö‡∞®‡±ç‡∞®‡±Ä ‡∞∏‡∞Æ‡∞æ‡∞®‡∞Ç‡∞ó‡∞æ ‡∞â‡∞®‡±ç‡∞®‡∞æ‡∞Ø‡∞æ ‡∞Ö‡∞®‡∞ø ‡∞ö‡±Ç‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
 * **‡∞´‡∞≤‡∞ø‡∞§‡∞Ç:** `True` ‡∞≤‡±á‡∞¶‡∞æ `False`.


### `__gt__(self, other)`


 * **‡∞é‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å ‡∞∞‡∞®‡±ç ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø:** `s1 > s2` ‡∞Ö‡∞®‡±ç‡∞®‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å.
 * **‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç:** ‡∞á‡∞ï‡±ç‡∞ï‡∞° "‡∞ó‡±ä‡∞™‡±ç‡∞™" ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ï‡±Å‡∞≤‡±Å ‡∞é‡∞ï‡±ç‡∞ï‡±Å‡∞µ ‡∞â‡∞Ç‡∞°‡∞ü‡∞Ç.
 * **‡∞´‡∞≤‡∞ø‡∞§‡∞Ç:** `True` ‡∞≤‡±á‡∞¶‡∞æ `False`.


### `__add__(self, other)`
